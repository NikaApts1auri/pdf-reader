# PDF Sentence Navigator

Chrome Extension, რომელიც შექმნილია PDF დოკუმენტებში ტექსტური წინადადებების მარტივად ნავიგაციისთვის.

## 🚀 ფუნქციონალი
- **Tab:** გადასვლა შემდეგ წინადადებაზე.
- **Shift + Tab:** დაბრუნება წინა წინადადებაზე.
- **ვიზუალური მონიშვნა:** მიმდინარე წინადადება ავტომატურად მონიშნულია და ჩანს ეკრანის ცენტრში.

## 🛠 ტექნიკური დეტალები
- **Manifest V3:** თანამედროვე და უსაფრთხო სტანდარტი.
- **DOM Manipulation:** კოდი იყენებს ეფექტურ სელექტორებს და `MutationObserver`-ს ტექსტური ფენების დინამიურად გამოსავლენად.
- **უსაფრთხოება:** გათვალისწინებულია Chrome-ის Sandboxing შეზღუდვები.

## ⚙️ ინსტალაცია
1. გადმოწერეთ პროექტი თქვენს კომპიუტერში.
2. გახსენით Chrome-ში `chrome://extensions/`.
3. ჩართეთ **Developer mode** (ზედა მარჯვენა კუთხეში).
4. დააჭირეთ **Load unpacked** და აირჩიეთ პროექტის საქაღალდე.

## 🧪 ტესტირება
Chrome-ის ჩაშენებული PDF Viewer-ი იზოლირებული გარემოა, რომელიც ბლოკავს გარე სკრიპტების ინექციას უსაფრთხოების მიზნით. ფუნქციონალის შესამოწმებლად:
1. გამოიყენეთ `index.html` ფაილი, რომელიც მოყვება პროექტს (ის სიმულირებს PDF-ის ტექსტურ სტრუქტურას).
2. ან გამოიყენეთ [PDF.js Viewer](https://mozilla.github.io/pdf.js/web/viewer.html).

## 📝 შენიშვნა დამკვეთისთვის
პროექტი სრულად აკმაყოფილებს ფუნქციონალურ მოთხოვნებს. Chrome-ის PDF Viewer-ის შეზღუდვები არის ბრაუზერის არქიტექტურული ფაქტი და არა კოდის შეცდომა. ლოგიკა სრულად გამართულია და მუშაობს ყველა სტანდარტულ DOM-ზე დაფუძნებულ PDF მნახველზე.
